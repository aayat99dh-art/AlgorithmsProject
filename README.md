# مشروع مادة الخوارزميات - سنة التخرج
## نظام إدارة سجلات الطلاب (Doubly Linked List)

هذا المشروع هو حل لمتطلبات الامتحان العملي، تم تطويره بلغة **C#** باستخدام بنية بيانات **اللائحة الخطية المزدوجة**. البرنامج مصمم ليتعامل مع سجلات الطلاب بطريقة مرنة وفعالة.

### تفاصيل المشروع التقنية:
* **بنية البيانات**: تم استخدام `Doubly Linked List` حيث تحتوي كل عقدة (Student) على مؤشرين `Next` و `Prev` للربط، مما يسمح لنا بالتنقل في اللائحة بالاتجاهين بسهولة.
* **أتمتة الحسابات والتقدير**: البرنامج يحسب المحصلة تلقائياً (متوسط علامتي الاختبار). كما يتم تحديد التقدير باستخدام `enum` (ممتاز، جيد جداً، جيد، راسب) بناءً على النتيجة النهائية.
* **البحث العودي (Recursive Search)**: تم تنفيذ خوارزمية البحث عن العلامات باستخدام المبدأ العودي (Recursion) بدلاً من الحلقات التكرارية، التزاماً بشروط المسألة.
* **خوارزمية الفرز**: طبقت خوارزمية **Bubble Sort** لفرز اللائحة إما حسب أسماء الطلاب (A-Z) أو حسب محصلاتهم من الأدنى للأعلى.



### وظائف القائمة الرئيسية:
1. **العرض**: إظهار قائمة كاملة بكل الطلاب مع محصلاتهم وتقديراتهم.
2. **الفرز**: ترتيب البيانات بطريقتين مختلفتين.
3. **البحث**: إيجاد الطلاب بناءً على علامات الاختبارات بطريقة عودية.
4. **الإضافة**: إمكانية إضافة طالب جديد في "بداية" أو "نهاية" اللائحة.
5. **الحذف**: إمكانية حذف سجل أي طالب من خلال رقمه الخاص (ID).
6. **المتفوقين**: ميزة إضافية لعرض الطلاب الذين حصلوا على محصلة أعلى من 85 فقط.

### ملاحظات إضافية:
* الكود منظم بطريقة تسهل القراءة مع تعليقات عربية تشرح وظيفة كل دالة.
* تم استخدام `try-catch` لضمان استقرار البرنامج وتجنب الانهيار عند إدخال بيانات خاطئة.
